# 修同質化的手法本身會同質化：均勻套用的修法複製出新模具

> **角色**：本卡是 `multi-round-review` 的支撐型原則、被「Round 2 cadence frame」與「反模式」段引用。
>
> **何時讀**：對批量 / 系列稿件修 cadence 同質化時、規劃修法形態與修法後重審時。

## 結論

為打破一個 cadence 模具而立的生成端規則、如果是「單一、均勻套用」的形態、會收斂出一個新模具、而且常比原模具更密。把「A 只說…；B 多了…」這種分號模具改成「段首一律目標詞先行」、套到整批、就長出「目標詞 + 抽象名詞 + 是/在」這個新模具——覆蓋率比原模具更高。

關鍵在自我感覺的遮蔽。套一條以「破模具 / 多樣化」為名的規則、在生成當下感覺像在製造變化、但那條規則本身就是一個單一模板。同源自審看不到、因為「我剛套了破模具規則」這件事替結果背了書。

## 為什麼均勻修法會複製出新模具

1. **換模不等於破模**：把模具 A 換成規則 B、若 B 只有一種實現形態、產出的就是模具 B。cadence 同質化的成因是「同位置逐字同句型」、不是用了哪個特定句型——換一個統一句型只是搬家。
2. **「多樣化規則」偽裝成多樣化**：規則的意圖（破模具）跟規則的形態（單一模板）是兩回事。作者盯著意圖、放過形態。
3. **同源自審被「已修」背書遮蔽**：剛執行完以「破模具」為名的動作、自審會預設結果更多樣、比原始盲區更難察覺——多了一層「我明明修過」的信心。

## 修法

- **輪替多個 framing、不換統一模板**：準備多個替代起手法（動詞先行 / 情境先行 / 對照先行 / 目標詞引號）、依序輪替分配。判準是修完後起手法有幾種、不是「有沒有避開舊句型」。
- **生成規則本身當 cadence 候選、進整組跨卡異源 review**：把新立的規則列進「這也可能同質化」的清單、修法後對整組跑一次 cadence frame（跨卡、同位置、逐字比對）。同源逐張自審在「已修好」錯覺下看不到、cadence 是跨卡屬性、必須整組看。
- **修法後的 cadence 重掃是固定工序**：「這批修完了」不是停止訊號、修法產物要重新過一次 cadence frame。

## 判讀徵兆

| 徵兆                               | 該做的行動                                          |
| ---------------------------------- | --------------------------------------------------- |
| 為破某模具立了「一律 X」的生成規則 | 預期 X 本身變新模具、修完對整組重掃 cadence         |
| 修法後感覺「有破模具、更多樣了」   | 這感覺是同源盲區背書、逐張看不算數、要跨卡逐字比對  |
| 修法只準備了一個替代句型           | 換模不是破模、準備多個 framing 輪替                 |
| cadence 只在單卡自審、沒跑整組異源 | cadence 是跨卡屬性、同源逐張看不到、要整組 reviewer |

# 描述抽象概念要用貼合屬性的謂語

> **角色**：本卡是 `compositional-writing` 的支撐型原則（principle）、被字句層 keyword bank 與 multi-pass review 輪 8-9（keyword bank + reader simulation）引用。
>
> **何時讀**：寫技術 / 分析文章時、判斷描述抽象概念（分析角度、框架、判讀訊號）的動詞與形容詞是真的貼合概念屬性、還是只是順口但語義錯位——尤其在字句層 register 掃描與異源冷讀時。

---

## 論述基礎與限制

本卡基於一次多輪審查中、兩處中文搭配問題由人類冷讀 catch 的事故。一批文章跑完 register / cadence / 冷讀多輪 agent review 都沒抓到兩處「順口但語義錯位」的搭配——把一個分析角度擬人化成「說了一半、還有話沒說完」的說話者、用「直接」形容訊號的可辨識度。兩處都由作者以外的人類當場指出。限制：case 是中文寫作，但原則（描述抽象概念要選屬性相符的謂語）跨語言成立；且它是 register 類問題、agent 同源自審對它有結構盲區。

---

## 核心原則

**描述抽象概念時，選跟該概念真實屬性相符的動詞與形容詞，不套用習慣上順口、但語義錯位的搭配。** 抽象概念（分析角度、框架、訊號、指標）沒有人的行為屬性、也沒有具體物的物理屬性——給它配上不相符的謂語，句子讀起來順、但語義是錯的。判別方法只有一個問題：這個謂語真的描述了這個名詞的屬性嗎，還是只是順口。順口是語感慣性、不是語義正確。

兩種常見錯配：

- **擬人化錯配**：把分析角度或框架當成會說話、有意圖的主體——「這個角度沒說完的話」「數字想告訴我們」「框架背後的潛台詞」。角度不會說話；它的屬性是涵蓋範圍，精確的說法是「這個角度沒有涵蓋 / 沒有提到的問題」。
- **形容詞錯配**：用習慣性但屬性不符的形容詞——「訊號很直接」。「直接」描述的是方式或態度（直接說、直接處理），不是訊號的可辨識度；訊號的可辨識度屬性用「清晰 / 明確 / 明顯」。

---

## 反模式：順口的搭配蓋過語義的正確

**Instance 1（擬人化）**：原句「這正是那個角度背後沒說完的話」。一個判讀角度被寫成「說了一半、還有話沒說完」的說話者。角度沒有「說」或「沒說完」這個行為屬性，它的屬性是「涵蓋了什麼、漏掉了什麼」。修正：「那個角度沒有提到的問題」。

**Instance 2（形容詞錯配）**：原句「判讀訊號很直接：……」。訊號的屬性是可辨識度（清不清楚、明不明顯），不是「直不直接」——「直接」形容的是行為的迂迴程度。修正：改用「訊號很明確」，或直接陳述訊號本身、拿掉這個評價性套語。

兩個 instance 的共同機制：作者走完思考、要下一個評價性的謂語時，抓了語感上最順的那個詞，而沒有回頭問它是否真的描述了主語的屬性。順口的搭配在同源自審裡讀起來完全自然——因為作者跟 agent reviewer 共享同一套語感慣性，這正是它需要異源冷讀的原因。

## 跟其他原則的關係

- [口語化修辭稀釋技術精度](colloquial-rhetoric-erodes-technical-precision.md)：同一取向、不同尺度——那條是整段語氣的口語化，本卡是單一詞對詞的搭配錯位；都是「順口壓過精確」，本卡更微觀。
- [避免濫用泛用詞](avoid-overused-generic-words.md)：都是用詞精確性、不同面向——那條是「一個泛用詞蓋多個不同情境」，本卡是「一個謂語配錯一個概念的屬性」。
- [教學散文的中性 register](teaching-prose-neutral-register.md)：本卡的搭配錯位無穩定關鍵詞、grep 抓不到、多輪同源 agent 自審全漏、靠人類異源冷讀 catch——跟中性 register 類違規同屬「同源盲區、真防線是文體異源視角」。

## 判讀徵兆

| 訊號                                                          | 該做的事                                                           |
| ------------------------------------------------------------- | ------------------------------------------------------------------ |
| 句子把一個角度／框架／數字寫成有意圖、會說話的主體            | 還原成該概念的屬性（涵蓋、遺漏、顯示），拿掉擬人化的動詞           |
| 用一個評價性形容詞形容抽象概念、讀起來順但停下來想不太對      | 問「這個詞真的描述了這個名詞的屬性嗎」，換成屬性相符的詞           |
| agent reviewer 多輪回報 register 乾淨、但總覺得某句「怪怪的」 | 這類搭配錯位無關鍵詞可掃、同源自審有盲區，交作者以外的人類冷讀定奪 |
